Use [[https://orgmode.org/manual/Easy-templates.html][Easy templates]] like so:


{{< highlight shell >}}
<s *TAB*
{{< /highlight >}}

And you get a nice orgmode source block.
{{< highlight org >}}
#+BEGIN_SRC
#+END_SRC
{{< / highlight >}}
